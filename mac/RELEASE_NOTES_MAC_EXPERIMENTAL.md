# Experimental Mac Support Release Notes

## 中文

本次更新新增 **Mac 手工安装版（实验性支持）**。

### 新增内容

- 新增 Mac 手工安装说明
- 新增 `install_mac.command` 一键安装脚本
- 新增 `restore_mac.command` 恢复脚本
- Release 新增 `zotero-word-links-mac-template.zip`
- 提供预改好的 `Zotero.dotm`
- Mac 版继续沿用相同按钮：
  - `Create Citation Links`
  - `Remove Citation Links`

### 适用范围

- `Zotero + Microsoft Word for Mac`

### 说明

- 当前为 **Experimental**
- 当前为 **One-click .command + Manual Fallback**
- 当前 **未在所有 Mac / Word 版本上完整验证**
- 不包含 Mac 自动安装器

### 风险提示

该模板基于 Zotero 官方 Mac Word 模板修改而来。  
如果你的 Zotero Word 集成版本不同，可能需要回滚到原始 `Zotero.dotm`。

## English

This update adds **experimental Mac manual-install support**.

### Added

- A Mac manual installation guide
- Added `install_mac.command` for one-click installation
- Added `restore_mac.command` for rollback
- A new release asset: `zotero-word-links-mac-template.zip`
- A prebuilt modified `Zotero.dotm`
- The same Word buttons are used on Mac:
  - `Create Citation Links`
  - `Remove Citation Links`

### Supported Scope

- `Zotero + Microsoft Word for Mac`

### Notes

- This is **Experimental**
- This is **One-click .command + Manual Fallback**
- It is **not yet fully validated across all Mac / Word versions**
- No automatic Mac installer is included

### Compatibility Note

This template is derived from Zotero's official Mac Word template.  
If your Zotero Word integration version differs, restore your original `Zotero.dotm` first.
